# 海房子食堂 · 每日三菜买菜助手

团队共享的食堂工具：每天 A/B/C 三栏抽取或点选菜品，按人数自动算出买菜量与大致价格，多人实时同步。

- **前端**：纯静态 `index.html`（无需构建）。
- **数据**：Supabase（菜谱 / 市价 / 今日三菜，实时同步）。
- **部署**：Netlify（发布目录＝仓库根目录，无构建命令）。

## 使用
1. 在 Supabase 跑一次建表 SQL。
2. 打开网页 → 「①菜库」点「🌱 载入默认菜谱」灌入初始数据。
3. 抽取 / 点选今日三菜 → 「✅ 确认」同步全员 → 「🧮 计算」出买菜清单。

## 交互
- 点菜名＝选用到今日；「🗑 删除模式」开启后点菜名才删除（防误删）。
- 菜谱按 3 人份录入，计算时按实际人数换算；重量向上取 0.5 斤、个/头向上取整。
